# Security Policy

## Supported versions

This project currently supports the latest commit on `main`.

## Reporting a vulnerability

Please **do not** open public issues for security vulnerabilities.

Use GitHub Security Advisories:

- https://github.com/XingCheng3/TextCreateImageTestView/security/advisories/new

Please include:

- Vulnerability description
- Reproduction steps / PoC
- Impact scope
- Suggested mitigation (if any)

We will acknowledge receipt as soon as possible and coordinate a fix + disclosure timeline.
